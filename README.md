# Feihong's ReScript quickstart

## Prerequisites

    wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
    nvm install 14
    nvm use 14

## Installation

    yarn install

## Commands

Start Parcel and ReScript compiler in watch mode:

    yarn start
